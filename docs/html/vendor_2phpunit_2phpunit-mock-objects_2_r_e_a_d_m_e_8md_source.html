5 # PHPUnit_MockObject
6 
7 **PHPUnit_MockObject** is the default mock object library for PHPUnit.
8 
9 ## Requirements
10 
11 * PHP 5.6 is required but using the latest version of PHP is highly recommended
12 
13 ## Installation
14 
15 You can add this library as a local, per-project dependency to your project using [Composer](https://getcomposer.org/):
16 
17  composer require phpunit/phpunit-mock-objects
18 
19 If you only need this library during development, for instance to run your project's test suite, then you should add it as a development-time dependency:
20 
21  composer require --dev phpunit/phpunit-mock-objects
