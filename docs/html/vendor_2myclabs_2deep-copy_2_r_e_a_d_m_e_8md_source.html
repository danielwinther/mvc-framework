50 ## Why?
51 
52 - How do you create copies of your objects?
53 
54 ```php
55 $myCopy = clone $myObject;
56 ```
57 
58 - How do you create **deep** copies of your objects (i.e. copying also all the objects referenced in the properties)?
59 
60 You use [`__clone()`](http://www.php.net/manual/en/language.oop5.cloning.php#object.clone) and implement the behavior yourself.
61 
62 - But how do you handle **cycles** in the association graph?
63 
64 Now you're in for a big mess :(

80 ## How it works
81 
82 DeepCopy recursively traverses all the object's properties and clones them. To avoid cloning the same object twice it keeps a hash map of all instances and thus preserves the object graph.
83 
84 ## Going further
85 
86 You can add filters to customize the copy process.
87 
88 The method to add a filter is `$deepCopy->addFilter($filter, $matcher)`,
89 with `$filter` implementing `DeepCopy\Filter\Filter`
90 and `$matcher` implementing `DeepCopy\Matcher\Matcher`.
91 
92 We provide some generic filters and matchers.
93 
94 ### Matchers
95 
96  - `DeepCopy\Matcher` applies on a object attribute.
97  - `DeepCopy\TypeMatcher` applies on any element found in graph, including array elements.
98 
99 #### Property name
100 
101 The `PropertyNameMatcher` will match a property by its name:
102 
103 ```php
104 use DeepCopy\Matcher\PropertyNameMatcher;
105 
106 $matcher = new PropertyNameMatcher('id');
107 // will apply a filter to any property of any objects named "id"
108 ```
109 
110 #### Specific property
111 
112 The `PropertyMatcher` will match a specific property of a specific class:
113 
114 ```php
115 use DeepCopy\Matcher\PropertyMatcher;
116 
117 $matcher = new PropertyMatcher('MyClass', 'id');
118 // will apply a filter to the property "id" of any objects of the class "MyClass"
119 ```
120 
121 #### Type
122 
123 The `TypeMatcher` will match any element by its type (instance of a class or any value that could be parameter of [gettype()](http://php.net/manual/en/function.gettype.php) function):
124 
125 ```php
126 use DeepCopy\TypeMatcher\TypeMatcher;
127 
128 $matcher = new TypeMatcher('Doctrine\Common\Collections\Collection');
129 // will apply a filter to any object that is an instance of Doctrine\Common\Collections\Collection
130 ```
131 
132 ### Filters
133 
134  - `DeepCopy\Filter` applies a transformation to the object attribute matched by `DeepCopy\Matcher`.
135  - `DeepCopy\TypeFilter` applies a transformation to any element matched by `DeepCopy\TypeMatcher`.
136 
137 #### `SetNullFilter`
138 
139 Let's say for example that you are copying a database record (or a Doctrine entity), so you want the copy not to have any ID:
140 
141 ```php
142 use DeepCopy\DeepCopy;
143 use DeepCopy\Filter\SetNullFilter;
144 use DeepCopy\Matcher\PropertyNameMatcher;
145 
146 $myObject = MyClass::load(123);
147 echo $myObject->id; // 123
148 
149 $deepCopy = new DeepCopy();
150 $deepCopy->addFilter(new SetNullFilter(), new PropertyNameMatcher('id'));
151 $myCopy = $deepCopy->copy($myObject);
152 
153 echo $myCopy->id; // null
154 ```
155 
156 #### `KeepFilter`
157 
158 If you want a property to remain untouched (for example, an association to an object):
159 
160 ```php
161 use DeepCopy\DeepCopy;
162 use DeepCopy\Filter\KeepFilter;
163 use DeepCopy\Matcher\PropertyMatcher;
164 
165 $deepCopy = new DeepCopy();
166 $deepCopy->addFilter(new KeepFilter(), new PropertyMatcher('MyClass', 'category'));
167 $myCopy = $deepCopy->copy($myObject);
168 
169 // $myCopy->category has not been touched
170 ```
171 
172 #### `ReplaceFilter`
173 
174  1. If you want to replace the value of a property:
175 
176  ```php
177  use DeepCopy\DeepCopy;
178  use DeepCopy\Filter\ReplaceFilter;
179  use DeepCopy\Matcher\PropertyMatcher;
180 
181  $deepCopy = new DeepCopy();
182  $callback = function ($currentValue) {
183  return $currentValue . ' (copy)'
184  };
185  $deepCopy->addFilter(new ReplaceFilter($callback), new PropertyMatcher('MyClass', 'title'));
186  $myCopy = $deepCopy->copy($myObject);
187 
188  // $myCopy->title will contain the data returned by the callback, e.g. 'The title (copy)'
189  ```
190 
191  2. If you want to replace whole element:
192 
193  ```php
194  use DeepCopy\DeepCopy;
195  use DeepCopy\TypeFilter\ReplaceFilter;
196  use DeepCopy\TypeMatcher\TypeMatcher;
197 
198  $deepCopy = new DeepCopy();
199  $callback = function (MyClass $myClass) {
200  return get_class($myClass);
201  };
202  $deepCopy->addTypeFilter(new ReplaceFilter($callback), new TypeMatcher('MyClass'));
203  $myCopy = $deepCopy->copy(array(new MyClass, 'some string', new MyClass));
204 
205  // $myCopy will contain ['MyClass', 'some stirng', 'MyClass']
206  ```
207 
208 
209 The `$callback` parameter of the `ReplaceFilter` constructor accepts any PHP callable.
210 
211 #### `ShallowCopyFilter`
212 
213 Stop *DeepCopy* from recursively copying element, using standard `clone` instead:
214 
215 ```php
216 use DeepCopy\DeepCopy;
217 use DeepCopy\TypeFilter\ShallowCopyFilter;
218 use DeepCopy\TypeMatcher\TypeMatcher;
219 use Mockery as m;
220 
221 $this->deepCopy = new DeepCopy();
222 $this->deepCopy->addTypeFilter(
223  new ShallowCopyFilter,
224  new TypeMatcher(m\MockInterface::class)
225 );
226 
227 $myServiceWithMocks = new MyService(m::mock(MyDependency1::class), m::mock(MyDependency2::class));
228 // all mocks will be just cloned, not deep-copied
229 ```
230 
231 #### `DoctrineCollectionFilter`
232 
233 If you use Doctrine and want to copy an entity, you will need to use the `DoctrineCollectionFilter`:
234 
235 ```php
236 use DeepCopy\DeepCopy;
237 use DeepCopy\Filter\Doctrine\DoctrineCollectionFilter;
238 use DeepCopy\Matcher\PropertyTypeMatcher;
239 
240 $deepCopy = new DeepCopy();
241 $deepCopy->addFilter(new DoctrineCollectionFilter(), new PropertyTypeMatcher('Doctrine\Common\Collections\Collection'));
242 $myCopy = $deepCopy->copy($myObject);
243 ```
244 
245 #### `DoctrineEmptyCollectionFilter`
246 
247 If you use Doctrine and want to copy an entity who contains a `Collection` that you want to be reset, you can use the `DoctrineEmptyCollectionFilter`
248 
249 ```php
250 use DeepCopy\DeepCopy;
251 use DeepCopy\Filter\Doctrine\DoctrineEmptyCollectionFilter;
252 use DeepCopy\Matcher\PropertyMatcher;
253 
254 $deepCopy = new DeepCopy();
255 $deepCopy->addFilter(new DoctrineEmptyCollectionFilter(), new PropertyMatcher('MyClass', 'myProperty'));
256 $myCopy = $deepCopy->copy($myObject);
257 
258 // $myCopy->myProperty will return an empty collection
259 ```
